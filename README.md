# Congruity Global Instability Engine, Planetary Seismic Instability Radar
Live site:
https://andrearomeo74-cloud.github.io/cgie-global-instability-radar/
Experimental public radar for detecting spatial clusters and systemic deviations in global seismic activity using public seismic data.

## Overview

This project presents an experimental seismic instability radar built on public earthquake data.

The system is designed to explore how clustered activity and deviations from systemic equilibrium may reveal emerging instability patterns across the global seismic network.

It is intended as a public prototype and exploratory observatory layer.

## What the radar does

The radar aggregates public seismic events into spatial cells and computes a simplified instability state based on:

, seismic energy  
, event density  
, systemic deviation from equilibrium  
, spatial clustering  
, dynamic instability phase

The output is a global radar map showing cells classified into instability phases.

## What this project is not

This project is not a deterministic earthquake prediction system.

It is an experimental instability detection framework for studying seismic activity as a complex dynamic system.

## Public prototype scope

This repository contains a simplified public implementation of the radar layer.

It does not expose the full internal logic of the broader Congruity framework.
