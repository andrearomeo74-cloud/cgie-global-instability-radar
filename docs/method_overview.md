# Method Overview

## Purpose

The purpose of this prototype is to explore seismic instability detection using a simplified dynamic equilibrium framework applied to public global seismic data.

## Core idea

Instead of treating earthquakes as isolated events, the radar treats seismic activity as part of a broader complex system.

Each spatial cell is evaluated in relation to a global equilibrium state derived from the current distribution of seismic indicators.

## Simplified indicators used in the public prototype

The current public prototype uses a simplified state representation based on:

, seismic energy  
, event concentration  
, local clustering  
, systemic deviation from equilibrium

## Dynamic interpretation

Each cell is classified into one of several qualitative system phases:

, Stable  
, Perturbation  
, Instability  
, Critical Cluster  
, Transition Risk

These phases describe how far a given region deviates from the current equilibrium of the global seismic system.

## Public scope

This public implementation is intentionally simplified.

It represents an observatory layer and does not include the full logic of the broader Congruity framework.

## Important note

This project is not intended as a deterministic earthquake prediction tool.

It is an experimental instability radar for research and exploratory analysis.
